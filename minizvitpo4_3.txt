Ось основні зміни та особливості цієї версії програми:

Програма обчислює інтеграл ∫[x^(n-1) * e^(-x)]dx від 0 до ∞ для наступних значень n:

n = 1/2
n = 1
n = 3/2
n = 2
n = (m+5)/2, де m - номер студента, який вводиться через інтерфейс


Використовується метод Монте-Карло для обчислення інтегралу.
Результати виводяться у текстовому полі для кожного значення n.
Графік показує порівняння результатів, отриманих методом Монте-Карло, з точними значеннями (обчисленими за допомогою функції math.gamma).
Користувач може ввести свій номер студента (m) через інтерфейс, і програма автоматично обчислить результат для n = (m+5)/2.

Ця версія програми точно відповідає вашому завданню. Вона обчислює інтеграл для всіх вказаних значень n, включаючи значення, що залежить від номера студента, і демонструє результати через графічний інтерфейс.
Щоб використати програму:

Запустіть код.
У вікні, що з'явиться, введіть свій номер студента (m).
Натисніть кнопку "Обчислити".
Програма виведе результати обчислень для всіх значень n та покаже графік порівняння з точними значеннями.

Ця програма дозволяє легко порівнювати точність методу Монте-Карло з точними значеннями для різних параметрів n, включаючи параметр, що залежить від номера студента.
