Цей код створює програму з графічним інтерфейсом для обчислення значень функції нормального розподілу для заданих параметрів. Ось короткий опис основних компонентів:

Функція integrand(u) визначає підінтегральну функцію e^(-u^2/2).
Функція f(x) обчислює значення заданої функції, використовуючи чисельне інтегрування за допомогою scipy.integrate.quad.
Клас NormalDistributionApp створює графічний інтерфейс з полем для введення номера студента (m), кнопкою для обчислення, текстовим полем для виведення результатів та областю для відображення графіка.
Метод calculate() виконує обчислення для x_i = 0.(i*m), де i змінюється від 0 до 10, та оновлює текстове поле і графік.

Щоб використати цю програму:

Запустіть код.
У вікні, що з'явиться, введіть номер студента (m).
Натисніть кнопку "Обчислити".
Програма виведе обчислені значення функції для різних x та покаже графік.

Ця програма дозволяє:

Обчислювати значення функції f(x) = 1/√(2π) * ∫[e^(-u^2/2)]du від 0 до x для різних x.
Візуалізувати результати у вигляді таблиці та графіка.
Легко змінювати параметр m (номер студента) для отримання різних наборів значень x.

Ви можете експериментувати з різними значеннями m, щоб побачити, як це впливає на результати обчислень та вигляд графіка. Зверніть увагу, що функція f(x) є кумулятивною функцією розподілу стандартного нормального розподілу, помноженою на 1/√(2π).
